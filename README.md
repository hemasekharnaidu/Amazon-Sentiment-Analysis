# Amazon Review Sentiment Analysis

A machine learning project that analyzes Amazon product reviews and classifies them as positive or negative using Logistic Regression.

## Features
- Text preprocessing
- Feature extraction
- Sentiment classification
- Model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Logistic Regression
- Google Colab

## Dataset
Amazon product reviews dataset used for training and testing the model.

## Results
Improved model accuracy from 72% to approximately 96.5%.
